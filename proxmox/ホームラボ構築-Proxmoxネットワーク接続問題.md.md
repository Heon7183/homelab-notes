# Proxmox初期構築時のネットワーク問題

## 目的
自宅の余っているデスクトップPCを利用して
ホームラボ環境を構築する。

## 実施内容
- Proxmoxのインストール
- 初期設定
- ネットワーク接続確認

## 発生した問題
Proxmox Web UIへの接続ができなかった。

## 原因
サーバーPCが有線ではなく、
USB Wi-Fiアダプタによる無線接続環境だったため。

Proxmoxでは一般的に
ブリッジ(vmbr0)を利用した有線環境での構成が前提となるケースが多く、
無線環境では正常に動作しなかった可能性がある。
